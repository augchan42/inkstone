# Create Verse Explanation

Invokes the **create-explanation** skill to produce a scholarly bilingual commentary on a classical Chinese verse.

## Output

- **English** explanation (100–150 words): literal scene → historical allusions → transformation meaning
- **繁體中文** explanation (150–200 chars): 文白相間
- **References**: historical figures, places, or allusions cited

## Usage

```
/inkstone:create-explanation 道生一，一生二，二生三，三生万物
```

Or paste any classical Chinese verse as the argument.
